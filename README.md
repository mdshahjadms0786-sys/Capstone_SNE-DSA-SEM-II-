# Capstone_SNE - DSA Semester II

## 📌 Project Aim
Social Network Explorer (Capstone Project for Data Structures Lab - ETCCDS202).

## 🔑 Features
- Hash Table for user management
- Trie for prefix search
- Graph with BFS/DFS for friendships
- Heap for trending posts
